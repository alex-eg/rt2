Build and run
-------------

Make sure you have [**SDL2**](https://www.libsdl.org/) library installed.

```
$ cd rt
$ cargo build --release
$ cargo run --release
```

Runnig without `--release` argument produces debug binaries, which are much slower and much more debuggable.

Controls
--------

* Q - roll CCW
* E - roll CW
* Right/Left arrow - yaw
* Up/Down arrow - pitch
* WASD - move camera
